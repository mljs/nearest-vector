nearestVector

src/index.js

Find the nearest vector in a list to a sample vector

nearestVector(listVectors: Array<Array<number>>, vector: Array<number>, options: [object]): (number | Array<number>)
Parameters
listVectors (Array<Array<number>>) List of vectors with same dimensions
vector (Array<number>) Reference vector to "classify"
options ([object]) Options object
Name Description
options.distanceFunction [function] (default squaredDistance) Function that receives two vectors and return their distance value as number
options.similarityFunction [function] (default undefined) Function that receives two vectors and return their similarity value as number
options.returnVector [boolean] (default false) Return the nearest vector instead of its index
Returns
(number | Array<number>): The index or the content of the nearest vector
